Install
------------------------------------------------------------------------------

``datasetLoader`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install datasetLoader

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade datasetLoader
